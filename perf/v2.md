# SIMD v2

This snapshot records the current accepted `src-simd2` state after the tri3-only
texture-locality optimisation. It builds on the accepted `v1` state.

## Accepted Optimisations In This Step

1. Keep the original hybrid texture sampler for higher-order and Newton-family kernels.
2. Add a separate tri3 / tri3opt-only hybrid sampler that:
   - compacts active lanes
   - orders active lanes by approximate texel locality
   - samples in that reordered lane order
   - writes results back to the original lane positions
3. Route only the low-order triangle textured SIMD perspective path through that sampler.

This avoids putting flags in the hot loop and avoids changing the higher-order
Newton-family texture path directly.

## Summary Vs `v1`

`fullraster`

- all-case average: about `+5.5%`
- tri3 / tri3opt average: about `+12.5%`
- higher-order / Newton-family average: about `+2.8%`
- strongest wins are low-order textured triangles:
  `tri3opt_tex8_grey_linear +31.2%`, `tri3_tex8_grey_linear +29.2%`,
  `tri3opt_tex8_rgb_linear +19.9%`, `tri3_tex8_grey_cubic +17.7%`

`geom`

- all-case average: about `+3.8%`
- tri3 / tri3opt average: about `+10.0%`
- higher-order / Newton-family average: about `+0.9%`
- strongest wins are again low-order textured triangles:
  `tri3_tex8_grey_linear +19.6%`, `tri3_tex8_grey_cubic +19.4%`,
  `tri3opt_tex8_grey_cubic +18.4%`

`sphere2000`

- all-case average: about `+3.2%`
- tri3 / tri3opt average: about `+9.0%`
- higher-order / Newton-family average: about `+1.0%`
- strongest wins:
  `tri3opt_tex8_grey +34.3%`, `tri3_tex8_grey +30.4%`,
  `tri3opt_tex8_rgb +13.1%`, `tri3_tex8_rgb +12.9%`

## Caveats

- Flat RGB tri3 cases regress slightly.
- Some higher-order RGB cases remain mixed, especially on `sphere2000`.
- `quad9_tex8_rgb` on `sphere2000` improved sharply in one run, so that case is worth
  rechecking if it becomes decision-critical.

## Overall Read

`v2` is worth keeping. The tri3-only texture locality split captures substantial wins on
the textured low-order triangle path without reproducing the broad regressions seen when
the same idea was applied across the higher-order and Newton-family kernels.
