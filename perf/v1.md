# SIMD v1

This snapshot records the current accepted `src-simd2` optimisation state against the
baseline SIMD implementation in `src-simd`.

## Accepted Optimisations

1. AoSoA Newton candidate storage while keeping the original three-pass Newton flow.
2. Improved Newton initial guess reuse based on recent converged inside-element seeds.

## Rejected Experiments

1. Newton immediate-shade / fused solve-and-shade path.
   This caused large regressions across the Newton kernels and was removed.
2. Triangle SIMD full-coverage / full-depth fast path.
   This was marginal and mixed, so it was removed.
3. Convex hull coarse-pass experiment.
   This caused correctness regressions on curved sphere cases and was removed.

## Summary Vs Baseline `src-simd`

`bench_fullraster`

- Newton-family average gain: about `+4.7%`
- Strongest gains are in `tri6`, especially:
  `tri6_flat_grey +17.8%`, `tri6_flat_rgb +16.1%`,
  `tri6_tex8_grey_linear +12.9%`, `tri6_tex8_rgb_linear +11.0%`
- `quad8` and `quad9` flat cases also improved:
  `quad8_flat_grey +8.4%`, `quad9_flat_grey +10.1%`
- There were no meaningful regressions in the Newton-family cases in this run.

`bench_geom`

- Newton-family average gain: about `+2.9%`
- Best gains:
  `tri6_flat_grey +9.5%`, `quad9_flat_rgb +9.3%`,
  `quad4newton_flat_grey +6.9%`, `quad8_tex8_rgb_linear +6.3%`,
  `tri6_tex8_grey_cubic +6.1%`
- Small regressions remain in some `quad9` grey-texture cases:
  `quad9_tex8_grey_cubic -2.1%`, `quad9_tex8_grey_linear -2.0%`,
  `quad9_tex8_grey_quintic -1.5%`

`bench_sphere2000`

- Newton-family average gain: about `+9.6%`
- This is the strongest benchmark family for the accepted optimisation set
- Largest gains:
  `quad4newton_flat_rgb +28.8%`, `quad4newton_flat_grey +19.2%`,
  `tri6_tex8_rgb +14.2%`, `quad9_tex8_rgb +14.0%`,
  `quad4newton_tex8_rgb +12.8%`
- Even the weaker cases still improved in this run; there were no actual regressions in
  the parsed Newton-family rows

`bench_sphere200`

- The accepted `src-simd2` benchmark was rerun and archived
- The baseline `src-simd` `sphere200` benchmark does not currently emit a top-level
  `benchmark.md`, so this file does not include a machine-parsed numeric comparison for
  that family

## Overall Read

The accepted SIMD v1 changes are worth keeping. The biggest gains are on the challenging
curved-element Newton workloads, especially `tri6` in `fullraster` and
`quad4newton` / `tri6` in `sphere2000`. `geom` is also net positive, though the gains
there are smaller and a few `quad9` grey-texture cases still regress slightly.
